Brand Setup — Design Spec

v2.0 · Orange + #111 + shadcn neutral

Layout

Single-column form card on zinc-50 background. Max-width 640px. Form sections separated by 1px borders inside the card. Footer has zinc-50 background to visually separate actions.

Key Design Changes

Brand avatar upload added (shows first letter of brand name as fallback)
Platform selection as clickable chip grid (not a multi-select dropdown)
API key fields have Show/Hide toggle
Form sections: Identity / Platforms / API Keys (clear grouping)
Input focus ring: orange (3px rgba spread, not indigo)
Delete brand button: red text, ghost by default, red border on hover
Required fields marked with red asterisk (not hidden)

Input Focus State

border-color: #fb923c (orange-400)
box-shadow: 0 0 0 3px rgba(251,146,60,0.12)

Same ring pattern as shadcnUI but with orange instead of indigo/blue. 12% opacity spread looks clean on white background.

Validation States

Error: Red border on input, red 12px message below. No toast — inline errors only.

Success (after save): Brief toast at bottom of screen: "Brand saved" with green check. Auto-dismiss 3s.

New Brand vs Edit

/brand/new: Page title "Set up your brand". No "Delete brand" button. No Cancel button.

/brand/[id]/edit: Page title "Edit brand". Shows "Delete brand" (destructive). Cancel returns to dashboard.
